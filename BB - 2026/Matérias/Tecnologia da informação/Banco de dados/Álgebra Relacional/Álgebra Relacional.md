Forma teórica de especificar operações sobre os dados usando o modelo relacional;

Entrada: uma ou duas relações;
Saída: uma nova relação, como resultado;

Operações unárias são sobre uma única relação.
Operações binárias são sobre duas relações.

Operações possíveis:
	- Seleção; (Unária)
	- Projeção; (Unária)
	- Renomeação; (Unária)
	- Atribuição; (Binária)
	- Produto cartesiano; (Binária)
	- Junção; (Binária)
	- Operações sobre conjuntos:
		- União; (Binária)
		- Diferença; (Binária)
		- Intersecção; (Binária)
